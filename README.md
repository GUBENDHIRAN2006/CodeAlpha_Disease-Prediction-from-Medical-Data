# Diabetes and Heart Disease Prediction using Machine Learning

This project is a Machine Learning-based healthcare prediction system designed to predict the likelihood of Diabetes and Heart Disease using patient medical data. The main objective of this project is to demonstrate how Machine Learning algorithms can assist in early disease detection and support healthcare decision-making through predictive analytics.

The system uses supervised learning techniques and data preprocessing methods to train highly accurate classification models. The project includes complete workflows such as data cleaning, feature selection, exploratory data analysis, feature scaling, model training, evaluation, and prediction.

For the Diabetes Prediction Model, medical attributes such as glucose level, BMI, insulin level, blood pressure, age, pregnancies, and diabetes pedigree function are used to predict whether a patient is diabetic or non-diabetic. For the Heart Disease Prediction Model, important health indicators like cholesterol, chest pain type, resting blood pressure, heart rate, and ECG-related features are used for prediction.

Several Machine Learning algorithms were tested and compared, including:

* Support Vector Machine (SVM)
* Logistic Regression
* Random Forest
* Decision Tree
* K-Nearest Neighbors (KNN)

Among them, optimized SVM models provided strong performance with improved accuracy after hyperparameter tuning and feature scaling using StandardScaler.

This project also includes:

* Model saving and loading using Joblib
* User input prediction system
* Data preprocessing pipeline
* Performance evaluation using Accuracy Score, Classification Report, and Confusion Matrix
* Clean and modular Python code

Technologies and Libraries Used:

* Python
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Joblib

This project helped me strengthen my understanding of Machine Learning workflows, healthcare analytics, model optimization, and real-world AI applications. It can also be extended into a web application using Flask, Streamlit, or Django for real-time disease prediction systems.
